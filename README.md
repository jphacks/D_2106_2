# Setup

1. clone this repository

```bash
$ git clone {this repository}
$ cd {this repository}
```

2. make .env file

```
DB_USER=<database user name>
DB_PASS=<database password>
DB_HOST=<database host>
DB_NAME=<database name>
DB_PORT=<database port>
```

3. build

```bash
$ docker compose build
```

4. run

```bash
$ docker compose run
```
